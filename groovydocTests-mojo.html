gplus:groovydocTests

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.6.2:groovydocTests

Description:

Generates GroovyDoc for the test sources.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Since version: 1.0-beta-1.

Optional Parameters

Name Type Since Description
displayAuthor boolean 1.0-beta-1 Whether to display the author in the generated GroovyDoc.
Default value is: true.
docTitle String 1.0-beta-1 The page title.
Default value is: Groovy Documentation.
footer String 1.0-beta-1 The page footer.
Default value is: Groovy Documentation.
groovyDocJavaSources boolean 1.0-beta-2 Whether to include Java sources in GroovyDoc generation.
Default value is: true.
groovyDocOutputDirectory File 1.0-beta-1 The location for the generated API docs.
Default value is: ${project.build.directory}/gapidocs.
header String 1.0-beta-1 The page header.
Default value is: Groovy Documentation.
links List 1.0-beta-2 Links to include in the generated GroovyDoc (key is link href, value is comma-separated packages to use that link).
overviewFile File 1.0-beta-1 The HTML file to be used for overview documentation.
scope String 1.0-beta-1 The scope to generate GroovyDoc for. Should be one of:
  • "public"
  • "protected"
  • "package"
  • "private"

Default value is: private.
skipGroovyDoc boolean 1.6 Flag to allow GroovyDoc generation to be skipped.
Default value is: false.
User property is: maven.groovydoc.skip.
skipTests boolean 1.0-beta-1 Flag to allow test compilation to be skipped.
Default value is: false.
User property is: maven.test.skip.
sources FileSet[] 1.0-beta-1 The Groovy source files (relative paths). Default: "${project.basedir}/src/main/groovy/**/*.groovy"
stylesheetEncoding String 1.0-beta-1 The encoding of stylesheetFile.
Default value is: ${project.build.sourceEncoding}.
stylesheetFile File 1.0-beta-1 The stylesheet file (absolute path) to copy to output directory (will overwrite default stylesheet.css).
testGroovyDocOutputDirectory File 1.0-beta-1 The location for the generated test API docs.
Default value is: ${project.build.directory}/testgapidocs.
testSources FileSet[] 1.0-beta-1 The Groovy test source files (relative paths). Default: "${project.basedir}/src/test/groovy/**/*.groovy"
windowTitle String 1.0-beta-1 The window title.
Default value is: Groovy Documentation.
