gplus:addTestSources

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.6.3:addTestSources

Description:

This mojo adds Groovy test sources to the project's test sources.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.0-beta-3.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
skipTests boolean 1.0-beta-3 Flag to allow test compilation/GroovyDoc/stub generation to be skipped.
Default value is: false.
User property is: maven.test.skip.
sources FileSet[] 1.0-beta-3 The Groovy source files (relative paths). Default: "${project.basedir}/src/main/groovy/**/*.groovy"
testSources FileSet[] 1.0-beta-3 The Groovy test source files (relative paths). Default: "${project.basedir}/src/test/groovy/**/*.groovy"
useSharedClassLoader boolean 1.6.3 Whether to use a shared classloader that includes both the project classpath and plugin classpath. Use only if you know what you're doing.
Default value is: false.
